We want to make Benny as helpful and useful as possible.

A tag prefix can be used to speed up the amount of time it takes to send a tag. For example, instead of sending 

!tag ExampleTag
you can use 
?ExampleTag
, just like seen here:

Discord User10/03/2020

?ExampleTag

BennyBot10/03/2020

This is an example tag

Note:

You can setup your tag prefix on your dashboard in the tags module.
