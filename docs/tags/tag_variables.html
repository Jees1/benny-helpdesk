Server Variables

Server Variables describe server information.

  • Server Name: 
    {server}
  • Server ID: 
    {server.id}
  • Server Icon: 
    {server.icon}
  • Server Owner: 
    {server.owner}
  • Server Region: 
    {server.region}

User Variables

User Variables describe information of the user who ran the tag command.

  • User Mention: 
    {user}
  • User Discriminator: 
    {user.tag}
  • User Avatar: 
    {user.avatar}
  • Username: 
    {user.name}
  • User ID: 
    {user.id}
Note: You can configure your tags by going to your dashboard and choosing the tags module.
