Features

Benny's tags module offers amazing features to help you even more. You can customize these options in your server's dashboard. You can view articles of these features here.

Permissions

For the Tags module to function correctly, it requires a list of permissions.

Configuration: Server Settings > Roles > Benny
Benny requires these permissions:
  • View Message History
  • View Messages
  • Send Messages
  • Embed Links
